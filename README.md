# Outpost

A unified self-hosted media server for movies, TV shows, music, and books.

## Features

- **Unified Library** - All your media in one place
- **Request System** - Request content with approval workflow
- **Discovery** - Find new content based on what you love
- **Multi-user** - User management with permissions
- **Metadata** - Automatic from TMDB, manual override available
- **Playback** - Stream with video/audio/subtitle selection
- **Self-hosted** - Your media, your server, your control

## Tech Stack

- **Frontend**: SvelteKit + TailwindCSS
- **Backend**: Go
- **Metadata**: TMDB API
