- Body: Regular, light gray
- Captions: Small, muted

## Components

### MediaCard

Poster card used in grids and rows.

Props:
- `type`: movie | show | album | book
- `id`: number
- `title`: string
- `year`: number (optional)
- `posterPath`: string
- `progress`: number (0-100, optional)
- `badge`: string (optional, e.g., "4K", "SERIES")

States:
- Default: Poster with title below
- Hover: Slight scale, play button appears
- With progress: Progress bar at bottom

### MediaRow

Horizontal scrolling row of MediaCards.

Props:
- `title`: string
- `items`: MediaCard[]
- `viewAllLink`: string (optional)

### SearchBar

Global search input.

Features:
- Debounced search (300ms)
- Dropdown results
- Navigate with arrow keys
- Enter to go to search page

### VideoPlayer

Full video player component.

Props:
- `src`: string (stream URL)
- `type`: movie | episode
- `mediaId`: number
- `title`: string
- `startPosition`: number (seconds)

Events:
- `onProgress`: (seconds) => void
- `onComplete`: () => void
- `onError`: (error) => void

### Modal

Reusable modal dialog.

Props:
- `open`: boolean
- `title`: string
- `size`: sm | md | lg | xl

### Toast

Notification toasts.

Types:
- success (green)
- error (red)
- warning (amber)
- info (blue)

## Loading States

- Skeleton loaders for cards
- Spinner for actions
- Progress bar for long operations

## Empty States

- Friendly message
- Action button when applicable
- Illustration (optional)

## Error States

- Clear error message
- Retry button when applicable
- Don't expose technical details to users
